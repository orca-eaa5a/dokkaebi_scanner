# DokEbin Scanner

![pic](./storage/pic.PNG)

## Description

Korean Hangul Word Document Formed Malware Scanner (Beta 0.1 v)



## Dependencies

* Python 3 >=
* PySide2 5.15.2 >=
* Yara (https://github.com/VirusTotal/yara/releases)



## Installing

```
1. git clone https://github.com/orca-eaa5a/dokkaebi_scanner.git

2. python setup.py install
```



## Executing program


```
python main.py
```
